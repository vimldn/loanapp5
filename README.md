# LoanApp.co.ke

Compare loan apps in Kenya - find the cheapest rates across M-Shwari, Tala, Branch, Fuliza, Hustler Fund, and 12+ other lenders.

## Features

- **Loan Calculator**: See exactly how much you'll pay back with each app
- **Comparison Table**: Sort by interest rate, max limit, term length, or rating
- **Filter by Type**: M-Pesa loans, Bank loans, or Fintech apps
- **Detailed Info**: Requirements, pros, cons, and CRB status for each app
- **Mobile Optimized**: Works great on phones

## Apps Included

- Hustler Fund (cheapest - 0.67%/month)
- M-Shwari
- Fuliza
- KCB M-Pesa
- Tala
- Branch
- Zenka
- Timiza (Absa)
- OKash
- Eazzy Loan (Equity)
- iPesa
- Kashway
